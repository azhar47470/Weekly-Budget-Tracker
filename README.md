# Weekly Budget Tracker

A Python program to track and analyze daily spending across a week.

## Features
- Records spending for each day of the week
- Calculates total and average weekly expense
- Finds highest spending day
- Shows days where spending stayed under $50

## How to Run
1. Make sure Python is installed
2. Run: python Weekly_Budget_Tracker.py

## Tech Used
Python — lists, loops, functions
